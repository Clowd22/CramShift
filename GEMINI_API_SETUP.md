# 🔐 Gemini API キーのセットアップガイド

## 📋 初回セットアップ手順

### **方法1: ローカルストレージを使用（ブラウザに保存）**

最も簡単な方法です。

1. **CramShift アプリケーションにアクセス**
2. **「⚙️ Gemini API キーの設定」セクションで、以下のAPIキーを入力：**

```
AIzaSyBHiqkOkvMJmVKAZHhu718HnL22arDhFvE
```

3. **「保存」ボタンをクリック**
4. **ブラウザに保存され、以降は自動で使用されます**

### **方法2: ローカルマシンの `.env` ファイル（開発環境用）**

`.env` ファイルはすでに作成済みです。確認してください：

```bash
# ファイルの場所
CramShift/.env
```

**内容:**
```
GEMINI_API_KEY=AIzaSyBHiqkOkvMJmVKAZHhu718HnL22arDhFvE
```

**⚠️ 重要:** `.env` ファイルは `.gitignore` に登録されているため、GitHub にはアップロードされません。

---

## 🔒 セキュリティに関する注意

- **`.env` ファイルは決して GitHub にコミットしないでください**
- **APIキーは個人情報であり、漏洩すると他者に悪用される可能性があります**
- **チーム開発の場合は、`.env.example` だけを共有し、実際のキーはメンバーが個々に設定**

## ❓ トラブルシューティング

**Q: 「Gemini API Error: 401」が表示される**
- A: APIキーが無効です。新しいキーを取得してください（[Google AI Studio](https://aistudio.google.com/apikey)）

**Q: 「Gemini API Error: 429」が表示される**
- A: APIリクエスト数が上限に達しました。しばらく待ってから再度試してください

**Q: ローカルストレージに保存されない**
- A: ブラウザのプライベートモード / シークレットモードでは使用できません。通常モードでアクセスしてください
